# Library-System
Kütüphane sistemi için gereken en iyi açık kaynak kodlu program


Bu program amatör düzeyde kütüphanelerin ihtiyacını karşılayabileceği ücretsiz özellikler sunmaktadır.
Herhangi bir alıntı, atıf vb. durumlar yoktur, Özgün programdır.
Kopyalanması yada ticarileştirilmesi halinde işlem başlatılacaktır.

Programın tam sürümü 1.0 dır ve sürekli güncelleme alacaktır.
